## 📌 Topics Covered

### 🔢 Ratio

#### 🧠 Concepts Learned

* If **a : b** and **b : c** are given, we can find:

  * **a : c**
  * **a : b : c**

#### 📘 Example

If
a : b = 2 : 3
b : c = 4 : 5

Make **b same**:

* a : b = 2 : 3 → multiply by 4 → 8 : 12
* b : c = 4 : 5 → multiply by 3 → 12 : 15

👉 Final:

* a : c = 8 : 15
* a : b : c = 8 : 12 : 15

### ⏳ Work & Time (Work / Day)

#### 🧠 Concepts Learned

* Work is inversely proportional to time
* Efficiency = Work per day

If:

* A completes work in 20 days → 1 day work = 1/20
* B completes work in 25 days → 1 day work = 1/25

#### 📘 Alternate Day Work Example

Day 1 → A works → 1/20
Day 2 → B works → 1/25

Work done in 2 days:
= 1/20 + 1/25
= (5 + 4) / 100
= 9/100

👉 In 2 days → 9/100 work

Remaining work:
= 91/100

Continue pattern until completion.

Final:

* 22 days → (11 × 9/100 = 99/100 work)
* Remaining = 1/100
* Next day (A works): finishes

👉 Total = **23 days**

### 🔀 Permutations

#### 🧠 Concepts Learned

1. **Permutation of n letters**

   * n!

2. **Permutation with repeating letters**

   * Formula:
     n! / (p! × q! × ...)

   Example:
   "AAB" → 3! / 2! = 3

3. **All vowels together**

   * Treat vowels as one unit
   * Arrange block + remaining letters

4. **With repetition allowed**

   * n^r

#### 📘 Example

Word: "APPLE"
= 5! / (2!)

---

### 🔢 Combinations

#### 🧠 Concepts Learned

* Selection (order does NOT matter)

#### 📘 Formula

nCr = n! / (r! × (n-r)!)

#### 📌 Similar to permutation but:

* Permutation → Arrangement
* Combination → Selection
